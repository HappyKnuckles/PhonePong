# Sound Files

This directory contains sound effects for the PhonePong game.

## Required Sound Files

Place the following sound files in this directory:

1. **ball_hit.mp3** - Played when the paddle hits the ball
2. **ball_bounce.mp3** - Played when the ball bounces on the table
3. **score.mp3** - Played when a point is scored
4. **game_start.mp3** - Played when the game starts

## Sound File Format

- **Format**: MP3 (recommended) or WAV
- **Sample Rate**: 44.1 kHz or 48 kHz
- **Bit Depth**: 16-bit or 24-bit
- **Duration**: Keep sounds short (< 1 second for hits/bounces, < 2 seconds for score/start)
- **Volume**: Normalize all files to the same peak level

## Where to Find Sounds

You can create or find free sound effects from:
- [Freesound.org](https://freesound.org/)
- [Zapsplat.com](https://www.zapsplat.com/)
- Record your own using a microphone and audio editing software

## Example Search Terms

- **Ball Hit**: "ping pong hit", "table tennis paddle", "bat hit"
- **Ball Bounce**: "ball bounce", "ping pong bounce", "table tennis bounce"
- **Score**: "point scored", "success", "win chime", "ding"
- **Game Start**: "game start", "ready go", "whistle", "countdown"

## Testing

After adding the sound files, test them in the game to ensure:
1. They play at the correct moments
2. Volume levels are appropriate
3. They don't overlap awkwardly
4. They enhance rather than distract from gameplay
